Basic Statistical Models

General Linear Model

  • Linear Regression
  • Logistic Regression
  • Response Variable: Continuous
  • Predictor Variables: Continuous/Categorical
  • Key Assumptions:
    • Linearity: The relationship between predictors and response is linear.
    • Independence: Observations are independent of each other.
    • Homoscedasticity: Constant variance of errors.
    • Normality: Errors (residuals) are normally distributed.
  • Violations & Remedies:
    • Non-linearity → Use polynomial terms or splines.
    • Heteroscedasticity → Use weighted least squares regression.
    • Non-normality of errors → Apply transformations (log, square root).
  • Statistical Inference:
    • Hypothesis tests (t-test, F-test)
    • Confidence intervals for regression coefficients
    • Analysis of Variance (ANOVA)
  • Estimation Method:
    • Ordinary Least Squares (OLS): Minimizes the sum of squared residuals.
  • Mathematical Expression: \[Y = \beta_0 + \sum \beta_i X_i + \epsilon, \quad \epsilon \sim N(0, \sigma^2)\] \[\hat{\beta} = (X^TX)^{-1}X^TY\]
  • Response Variable: Binary (0/1)
  • Predictor Variables: Continuous/Categorical
  • Key Assumptions:
    • Linearity in log-odds: The logit transformation of the probability of success is linearly related to the predictors.
    • Independence: Observations are independent.
    • No multicollinearity: Predictors should not be highly correlated.
    • Large sample size: Asymptotic properties improve inference.
  • Violations & Remedies:
    • Non-linearity in log-odds → Use polynomial terms, splines, or interaction terms.
    • Multicollinearity → Remove or combine highly correlated variables (VIF check).
    • Separation issue → Use penalized regression (e.g., LASSO, ridge).
  • Statistical Inference:
    • Wald Test, Likelihood Ratio Test
    • Odds Ratios
    • Confidence Intervals for Coefficients
  • Estimation Method:
    • Maximum Likelihood Estimation (MLE)
  • Mathematical Expression: \[P(Y=1) = \frac{e^{\beta_0 + \sum \beta_i X_i}}{1 + e^{\beta_0 + \sum \beta_i X_i}}\] \[\log \left( \frac{P(Y=1)}{1 - P(Y=1)} \right) = \beta_0 + \sum \beta_i X_i\]
